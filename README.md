الذكاء الاصطناعي (AI): المظلة الكبرى التي تشمل أي آلة تحاكي السلوك الذكي للبشر.
التعلم الآلي (Machine Learning): نظام فرعي يركز على تدريب الآلة لتتعلم من البيانات وتتطور تلقائياً. يمكن معرفة المزيد عبر صفحة Oracle الإمارات العربية المتحدة.
